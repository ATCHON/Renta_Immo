# Story S5.3: Comparaison des Régimes Fiscaux

**ID** : S5.3  
**Titre** : Comparateur de régimes côte à côte  
**Priorité** : P1  
**Statut** : ✅ TERMINÉ (Intégré Janv 2026)

## 1. Objectif
Permettre à l'investisseur de comparer instantanément l'impact de différents régimes fiscaux (LMNP Réel, Micro-BIC, SCI IS, etc.) sur le même bien afin de choisir la stratégie la plus rentable.

## 2. Critères d'acceptation
- [x] Affichage d'un tableau comparatif des 5 régimes principaux :
    - Location nue (Micro-foncier)
    - Location nue (Foncier réel)
    - LMNP (Micro-BIC)
    - LMNP (Réel)
    - SCI à l'IS (Capitalisation & Distribution)
- [x] Comparaison sur 3 indicateurs clés :
    - Impôt moyen annuel (sur 10 ans)
    - Cashflow net annuel moyen
    - Rentabilité nette-nette
- [x] Mise en évidence (badge "Optimal") du régime offrant le meilleur cashflow net.
- [x] Tooltip pédagogique expliquant pourquoi un régime est plus avantageux qu'un autre dans le cas présent.

## 3. Impact technique
- **Frontend** : Création d'un nouveau composant `FiscalComparator` dans la page de résultats.
- **Calculs** : Les fonctions de calcul existent déjà dans `fiscalite.ts`, il faut les orchestrer pour générer les 5 variantes en une fois (ou à la demande).
- **UI** : Intégration dans le dashboard de résultats.
