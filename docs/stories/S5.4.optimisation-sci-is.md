# Story : Optimisation Fiscale SCI IS & LMNP Réel

**ID** : S5.4  
**Titre** : Déduction des frais financiers  
**Priorité** : P2  
**Statut** : Validé (Intégré Janv 2026)

## 1. Objectif
Fiabiliser l'estimation fiscale en déduisant les intérêts d'emprunt et l'assurance de prêt de la base imposable pour les régimes au réel (LMNP Réel, SCI IS, Foncier Réel).

## 2. Critères d'acceptation
- [x] Pour le régime **SCI à l'IS**, la base imposable est : Loyer - Charges - Amortissement - Intérêts - Assurance.
- [x] Pour le régime **LMNP Réel**, même logique de déduction des frais financiers.
- [x] Pour le **Foncier Réel**, déduction des intérêts et assurance.
- [x] Implémentation de la Flat Tax (30%) pour la distribution de dividendes en SCI IS.

## 3. Impact technique
- `src/server/calculations/fiscalite.ts` : les fonctions de calcul doivent recevoir le montant annuel des intérêts et assurances.
- `src/server/calculations/index.ts` : orchestrer le passage de ces données du module financement vers le module fiscalité.
