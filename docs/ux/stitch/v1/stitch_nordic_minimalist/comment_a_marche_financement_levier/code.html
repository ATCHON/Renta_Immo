Comprendre le Levier Bancaire

Mensualité de Crédit (Formule PMT)

La mensualité est calculée selon une formule d'annuité constante :

M = C x t x (1+t)^n / ((1+t)^n - 1) + Assurance
M : Mensualité totale
C : Capital emprunté
t : Taux mensuel (Annuel / 12)
n : Nombre de mensualités
lightbulb

Le Concept de Levier

Emprunter à 3.5% pour investir dans un actif qui rapporte 6% net, c'est empocher une marge de 2.5% sur de l'argent que vous n'avez pas. C'est l'accélérateur de patrimoine par excellence.

Assurance Emprunteur

Capital Initial (Classique)

Assurance/mois = Capital emprunté x Taux / 12

Mensualité Constante

50 € / mois fixe

Mode par défaut des contrats groupe bancaires.

Capital Restant Dû (CRD)

Assurance mois N = Capital restant(N) x Taux / 12

Mensualité Dégressive

50 € arrow_forward 2.5 € / mois

Économie de ~40% sur le coût total du crédit.

info

En mode CRD, l'économie améliore votre cash-flow en fin de crédit et augmente votre TRI. Cependant, pour le calcul du taux d'endettement, les banques retiennent souvent la mensualité de l'année 1.
